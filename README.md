A group-based ping/notification bot for Reddit communities built using Devvit.
Allows users to subscribe to interest groups and ping group members via comment commands, with full moderator controls, blacklist support, and persistent wiki-backed storage..
Features:
\nGroup-based ping system (!ping <group>)
\nSubscribe / Unsubscribe via comments
\nPersistent storage using subreddit wiki
\nModerator blacklist (via mod menu)
\nPing usage statistics per group
\nConfigurable groups via app settings (up to 12)
\nMods-only wiki access
\nCase-insensitive usernames
